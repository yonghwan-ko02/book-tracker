# 📚 LuminaRead — AI Book Tracker & Cart

LuminaRead(루미나 리드)는 바닐라 자바스크립트(Vanilla JS)와 Supabase를 기반으로 한 **프리미엄 3중 백업 도서 검색, AI 기반의 독서 취향 분석, 독서 습관 기록 및 노션 포트폴리오 연동 서비스**입니다. 

화려한 다크 글래스모피즘(Glassmorphism) 테마와 HSL 네온 컬러 시스템을 채택하여 최고의 시각적 만족과 부드러운 반응형 웹 서재 관리를 지원합니다.

👉 **로컬 개발 서버 구동 주소**: **[http://localhost:5173/](http://localhost:5173/)**

---

## ✨ 핵심 차별점 & 주요 기능

### 1. 🛡️ 3중 안심 도서 검색 파이프라인 (Resilient Search Pipeline)
Google Books API 서버의 과도한 호출 차단(429 Rate Limit)이나 일시적인 네트워크 장애가 감지되면, 시스템이 정지되지 않고 다음 백업 소스로 순차 우회 작동합니다.
- **1단계**: Google Books OpenAPI 실시간 검색.
- **2단계**: 장애 시, **CORS 개방형 Open Library OpenAPI**로 즉각 실시간 전환 검색.
- **3단계**: 통신 단절 시, 내장된 **분야별 베스트셀러 로컬 DB** 필터링 검색 작동.

### 2. 🧠 Lumina AI 맞춤 도서 추천 비서
- 대시보드 화면에서 사용자의 최근 독서 이력 장르 비율, 완독 여부, 별점 평가 데이터를 실시간 분석합니다.
- 사용자만을 위한 개별 **"AI 성향 비평 코멘트"**를 제공하며 취향을 저격할 3종 도서를 추천합니다.
- 추천 도서는 대시보드에서 클릭 한 번으로 **내 서재에 등록**하거나 **장바구니에 적재**할 수 있습니다.

### 3. 📝 노션(Notion) 독서 포트폴리오 & CSV 내보내기 센터
- **CSV 내보내기**: 한글 깨짐이 완벽히 방지된 UTF-8 BOM 인코딩(`\uFEFF`) 파일로 서재 정보를 다운로드하여 노션 데이터베이스에 바로 업로드할 수 있습니다.
- **노션 포트폴리오 복사**: 현재 읽은 도서 평점 보드(표 형식)와 일자별 데일리 독서 로그, 생각 기록 일지 전체를 노션 친화적 마크다운으로 클립보드에 복사해 줍니다. 노션 문서에 그대로 `Ctrl+V`를 누르면 즉시 포트폴리오가 완성됩니다.

### 4. 🔓 테스트용 0.1초 자동 프리패스 로그인 (Silent Auto-Login)
- 귀찮은 이메일 가입 및 인증 절차 없이, 사이트에 접속하자마자 백그라운드에서 데모용 계정(`demo@luminaread.com` / `password123`)으로 **자동 로그인이 실시간 수행**됩니다.
- 로그인 모달 방해막 없이 즉각 모든 서재와 대시보드, 타임라인 기능을 풀플로우로 테스트해 볼 수 있습니다.

### 5. 🛒 장바구니 및 국내 대형 서점(교보문고, 알라딘) 통합 검색 연동
- 장바구니에 보관한 도서 목록을 바탕으로 모의 정가를 연산합니다.
- 결제 버튼 클릭 시 책의 고유 ISBN 번호(또는 제목)를 활용하여 **교보문고 실시간 도서 상세 검색창** 및 **알라딘 도서 상세 검색창** 멀티 탭이 즉각 브라우저에 연동되어 실구매로 매끄럽게 연결됩니다.

## 🚀 로컬 실행 방법 (Quick Start)

### 1) 패키지 설치 및 서버 구동
Vite가 장착되어 있어 1초 내로 개발 서버가 가동됩니다.
```bash
# 1. 패키지 의존성 설치
npm install

# 2. 로컬 Vite 개발 서버 실행
npm run dev
```
구동이 완료되면 브라우저에서 **`http://localhost:5173/`**로 접속합니다.

## 🛢️ Supabase 백엔드 데이터베이스 구축 정보 (참고용)
이미 연결이 완료되어 있으나, 새로운 Supabase 프로젝트에 직접 세팅하길 원하신다면 `src/sql/schema.sql` 파일의 쿼리를 Supabase 대시보드 [SQL Editor]에 기입해 실행하신 후 `src/js/supabase.js` 파일의 연동 키값만 갱신하시면 됩니다.

1. **테이블 생성**: `books`, `reading_logs`, `cart_items`
2. **이메일 즉시 승인 트리거**: 회원가입 시 번거로운 이메일 인증 절차를 건너뛰는 트리거 세팅.
3. **RLS (Row Level Security) 설정**: 사용자가 자신이 저장한 책과 로그에만 권한을 가질 수 있도록 철저한 유저 아이디 기준 정책 활성화.
4. **역할 권한 부여 (Grants)**: PostgREST API를 통한 프론트엔드 연결 시 `authenticated`, `anon` 권한에 완벽한 테이블 접근 권한 부여.
